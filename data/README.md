# Data

This repo follows a **safe** approach:
- The full survey file is **not committed** to GitHub.
- The repo includes **small samples** for preview and reproducibility.

## Local setup
Place the full file here (local only):
- `data/raw/survey_results_public.csv`

The original raw file contains **53,853 responses** across **177 countries**.

## Included in this repo
- `data/sample/survey_results_schema.csv`
- `data/sample/survey_results_public_sample.csv`
- `data/sample/so2025_tableau_clean_final_sample.csv`

## Outputs generated locally
